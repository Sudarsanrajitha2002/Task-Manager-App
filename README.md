## Overview 
Full-stack task management application with team collaboration features. 
 
## Features 
- Task creation and assignment 
- Priority management 
- Deadline tracking 
- Team collaboration 
- Progress monitoring 
 
## Tech Stack 
- **Backend:** Node.js, Express.js 
- **Database:** MongoDB 
- **Frontend:** React.js
